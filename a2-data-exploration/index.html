Question: What is the trend of the traffic violation by years caused by different race?

Visual Encoding

In order to compare the difference between each year's traffic violation by the different race, I found bar chart is the best choice to do the comparison. It is easy for users to pick up the information by comparing the position in common scale with different years. By clicking on the button on top of the graph, it switches to a different race, so users are also being able to judge the difference between races by comparing the length in common scale that the same y-axis value they share. Also, by hover on each bar, the graph provide graph information for users to easily get the exact number. Also, I found out that there are many wrong recorded data such the year is later than 2017 or other weird numbers, so I chose to control the number to be only between 1800 to 2017 currently.

Control Features

Users will be able to switch different races by clicking the button on the top of the graph. When clicking to a specific race, the graph displaces the same race's traffic violation in different years from daily record. With six different buttons to switch the race, the users will be able to compare the race different for the traffic violation, bot only the yearly difference by the visual encoding.
